# Task 3 - Data Visualization

## Objective
The goal of this project is to transform raw population data into meaningful visual insights using Power BI and Python.

## Tools Used
- Python
- Pandas
- Power BI
- Matplotlib
- Seaborn

## Dashboard Features
- KPI Cards for population metrics
- Top populated countries analysis
- Fertility rate vs median age scatter analysis
- Population density distribution visualization

## Key Insights
- India and China dominate global population.
- Countries with higher fertility rates generally have lower median ages.
- Most countries have relatively low population density, while only a few are highly dense.
