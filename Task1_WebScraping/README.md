To extract population data from a public website using Python web scraping techniques.

## Tools Used
- Python
- BeautifulSoup
- Requests
- Pandas
- Google Colab

https://www.worldometers.info/world-population/population-by-country/

## Output
- population_data.csv

## Features
- Extracted HTML table data
- Converted data into DataFrame
- Exported structured CSV dataset
