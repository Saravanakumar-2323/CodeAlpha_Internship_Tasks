# Task 2 - Exploratory Data Analysis (EDA)

## Objective
To analyze and understand the population dataset using Exploratory Data Analysis techniques.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data structure exploration
- Missing value analysis
- Statistical summary
- Trend and pattern identification
- Anomaly detection
- Hypothesis testing
- Correlation analysis using heatmap

## Key Insights
- India and China have the highest populations.
- Population density varies significantly across countries.
- Land area alone does not determine population size.
- Data cleaning was required for numeric analysis.
